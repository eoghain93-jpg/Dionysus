# Refunds Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Cashiers can refund a completed order from today's transactions. The refund requires PIN confirmation via a `<PinGate>` component. Payment reversal is automatic per method: cash shows a "Hand back £X.XX" prompt, card calls a Stripe refund via the `process-refund` edge function, and tab credits the amount back to the member's `tab_balance`.

**Architecture:** A new `RefundsModal` component lists today's completed non-refunded orders fetched from Supabase. Selecting an order shows its items and total. A mocked `<PinGate>` confirms the action. Post-PIN, the flow branches by payment method. A new `process-refund` Supabase edge function handles card refunds (Stripe API) and tab credits (RPC), and writes `refunded_at`/`refunded_by` to the `orders` table. A DB migration adds those two columns. The "Refund" button is added to `OrderPanel` below the existing payment buttons.

**Tech Stack:** React + Vite, Vitest + @testing-library/react, Tailwind CSS, lucide-react, Zustand, Supabase JS v2 (browser), Supabase Edge Functions (Deno), Stripe API v14

## Task 1: DB migration — `refunded_at` and `refunded_by` columns

**Files:**
- Create: `supabase/migrations/20260330_refunds.sql`

### Step 1: Create the migration file

```sql
-- supabase/migrations/20260330_refunds.sql

-- Add refund tracking columns to orders
alter table orders add column if not exists refunded_at timestamptz;
alter table orders add column if not exists refunded_by uuid references members(id);

-- Index for fast "find refunded orders" queries in Z report
create index if not exists orders_refunded_at_idx on orders(refunded_at) where refunded_at is not null;

-- RPC: credit tab balance back to a member atomically (used by process-refund edge function
-- and potentially by future direct-tab-refund flows)
create or replace function credit_tab_balance(p_member_id uuid, p_amount numeric)
returns void language plpgsql security definer as $$
begin
  update members
  set tab_balance = tab_balance + p_amount
  where id = p_member_id;

  if not found then
    raise exception 'Member % not found', p_member_id;
  end if;
end;
$$;
```

## Task 2: `process-refund` edge function

**Files:**
- Create: `supabase/functions/process-refund/index.ts`

This edge function accepts `{ order_id, staff_member_id }`, verifies the order exists and is not already refunded, performs the payment reversal (Stripe for card, RPC for tab, no-op for cash), stamps `refunded_at`/`refunded_by`, and returns `{ success: true }` or `{ error }`.

### Step 1: Write the edge function

```typescript
// supabase/functions/process-refund/index.ts
import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'
import Stripe from 'https://esm.sh/stripe@14'

const corsHeaders = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
}

serve(async (req) => {
  if (req.method === 'OPTIONS') {
    return new Response('ok', { headers: corsHeaders })
  }

  if (req.method !== 'POST') {
    return new Response('Method not allowed', { status: 405, headers: corsHeaders })
  }

  // Verify the request carries a valid Supabase JWT
  const authHeader = req.headers.get('Authorization')
  if (!authHeader?.startsWith('Bearer ')) {
    return new Response(JSON.stringify({ error: 'Unauthorized' }), {
      status: 401,
      headers: { 'Content-Type': 'application/json', ...corsHeaders },
    })
  }

  // Verify the JWT is valid
  const anonClient = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_ANON_KEY')!,
    { global: { headers: { Authorization: authHeader } } }
  )
  const { error: jwtError } = await anonClient.auth.getUser()
  if (jwtError) {
    return new Response(JSON.stringify({ error: 'Unauthorized' }), {
      status: 401,
      headers: { 'Content-Type': 'application/json', ...corsHeaders },
    })
  }

  let order_id: string, staff_member_id: string
  try {
    const body = await req.json()
    order_id = body.order_id
    staff_member_id = body.staff_member_id
  } catch {
    return new Response(JSON.stringify({ error: 'Invalid JSON body' }), {
      status: 400,
      headers: { 'Content-Type': 'application/json', ...corsHeaders },
    })
  }

  if (!order_id || !staff_member_id) {
    return new Response(JSON.stringify({ error: 'order_id and staff_member_id required' }), {
      status: 400,
      headers: { 'Content-Type': 'application/json', ...corsHeaders },
    })
  }

  const supabase = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!
  )

  // Fetch the order
  const { data: order, error: orderError } = await supabase
    .from('orders')
    .select('id, total_amount, payment_method, stripe_payment_intent_id, member_id, refunded_at')
    .eq('id', order_id)
    .single()

  if (orderError || !order) {
    return new Response(JSON.stringify({ error: 'Order not found' }), {
      status: 404,
      headers: { 'Content-Type': 'application/json', ...corsHeaders },
    })
  }

  // Idempotency guard — do not double-refund
  if (order.refunded_at) {
    return new Response(JSON.stringify({ error: 'Order already refunded' }), {
      status: 409,
      headers: { 'Content-Type': 'application/json', ...corsHeaders },
    })
  }

  // Payment reversal by method
  if (order.payment_method === 'card') {
    if (!order.stripe_payment_intent_id) {
      return new Response(JSON.stringify({ error: 'No Stripe payment intent on order' }), {
        status: 422,
        headers: { 'Content-Type': 'application/json', ...corsHeaders },
      })
    }

    const stripe = new Stripe(Deno.env.get('STRIPE_SECRET_KEY')!, { apiVersion: '2023-10-16' })

    try {
      await stripe.refunds.create({ payment_intent: order.stripe_payment_intent_id })
    } catch (err) {
      const message = err instanceof Error ? err.message : 'Stripe refund failed'
      return new Response(JSON.stringify({ error: message }), {
        status: 502,
        headers: { 'Content-Type': 'application/json', ...corsHeaders },
      })
    }
  }

  if (order.payment_method === 'tab') {
    if (!order.member_id) {
      return new Response(JSON.stringify({ error: 'Tab order has no member_id' }), {
        status: 422,
        headers: { 'Content-Type': 'application/json', ...corsHeaders },
      })
    }

    const { error: rpcError } = await supabase.rpc('credit_tab_balance', {
      p_member_id: order.member_id,
      p_amount: order.total_amount,
    })

    if (rpcError) {
      return new Response(JSON.stringify({ error: rpcError.message }), {
        status: 500,
        headers: { 'Content-Type': 'application/json', ...corsHeaders },
      })
    }
  }

  // Cash refunds: no server-side action — cashier hands back cash manually.
  // Record is stamped below in all cases.

  // Stamp the refund
  const { error: updateError } = await supabase
    .from('orders')
    .update({ refunded_at: new Date().toISOString(), refunded_by: staff_member_id })
    .eq('id', order_id)

  if (updateError) {
    return new Response(JSON.stringify({ error: updateError.message }), {
      status: 500,
      headers: { 'Content-Type': 'application/json', ...corsHeaders },
    })
  }

  return new Response(JSON.stringify({ success: true }), {
    status: 200,
    headers: { 'Content-Type': 'application/json', ...corsHeaders },
  })
})
```

## Task 3: `RefundsModal` component

**Files:**
- Create: `src/components/till/RefundsModal.jsx`
- Create: `src/components/till/RefundsModal.test.jsx`

`RefundsModal` is the main refund UI. It:
1. On mount, fetches today's completed non-refunded orders from Supabase.
2. Shows a list of those orders (time, items summary, total, payment method badge).
3. When an order is selected, shows its details and a "Refund" confirm button gated by `<PinGate>`.
4. After PIN confirmation, branches by payment method:
   - **Cash** → shows a "Hand back £X.XX" confirmation screen with a "Done" button that records the refund.
   - **Card** → shows a loading state, calls `supabase.functions.invoke('process-refund', ...)`, then shows success or error.
   - **Tab** → same as Card (process-refund handles the RPC credit).
5. On success, calls the `onRefunded` callback and closes.

Props: `onClose: () => void`, `onRefunded: () => void`, `staffMemberId: string`

```jsx
/**
 * RefundsModal
 *
 * Props:
 *   onClose       () => void   — called when the modal should close
 *   onRefunded    () => void   — called after a successful refund (before onClose)
 *   staffMemberId string       — UUID of the currently logged-in staff member
 */
export default function RefundsModal({ onClose, onRefunded, staffMemberId }) {
  const [orders, setOrders] = useState([])
  const [loading, setLoading] = useState(true)
  const [fetchError, setFetchError] = useState(null)

  // Selected order and UI step
  const [selectedOrder, setSelectedOrder] = useState(null)
  // step: 'list' | 'detail' | 'pin' | 'cash-confirm' | 'processing'
  const [step, setStep] = useState('list')
  const [processing, setProcessing] = useState(false)
  const [refundError, setRefundError] = useState(null)

  // ── Fetch today's orders on mount ──────────────────────────────────────────
  useEffect(() => {
    async function load() {
      setLoading(true)
      setFetchError(null)
      const { data, error } = await supabase
        .from('orders')
        .select('id, created_at, total_amount, payment_method, status, refunded_at, stripe_payment_intent_id, member_id, order_items(id, name, quantity, unit_price)')
        .eq('status', 'paid')
        .gte('created_at', todayStart())
        .order('created_at', { ascending: false })

      if (error) {
        setFetchError('Failed to load orders. Please try again.')
      } else {
        // Only show non-refunded orders
        setOrders((data ?? []).filter(o => !o.refunded_at))
      }
      setLoading(false)
    }
    load()
  }, [])

  // ── Handlers ───────────────────────────────────────────────────────────────

  function handleSelectOrder(order) {
    setSelectedOrder(order)
    setRefundError(null)
    setStep('detail')
  }

  function handleRefundClick() {
    setRefundError(null)
    setStep('pin')
  }

  function handlePinCancel() {
    setStep('detail')
  }

  function handlePinConfirm() {
    if (selectedOrder.payment_method === 'cash') {
      setStep('cash-confirm')
    } else {
      // Card and tab go straight to processing
      processRefund()
    }
  }

  function handleCashCancel() {
    setStep('detail')
  }

  async function handleCashDone() {
    await processRefund()
  }

  async function processRefund() {
    setProcessing(true)
    setRefundError(null)
    if (selectedOrder.payment_method !== 'cash') {
      setStep('processing')
    }

    const { data, error } = await supabase.functions.invoke('process-refund', {
      body: { order_id: selectedOrder.id, staff_member_id: staffMemberId },
    })

    setProcessing(false)

    if (error || !data?.success) {
      const msg = error?.message ?? data?.error ?? 'Refund failed. Please try again.'
      setRefundError(msg)
      if (selectedOrder.payment_method !== 'cash') {
        // Stay on processing screen to show the error
      }
      return
    }

    onRefunded()
    onClose()
  }

  // ── Render ─────────────────────────────────────────────────────────────────

  return (
    <div className="fixed inset-0 bg-black/70 flex items-center justify-center z-50 p-4">
      <div className="bg-slate-800 rounded-2xl w-full max-w-md max-h-[90vh] flex flex-col">
        {/* Header */}
        <div className="flex items-center justify-between px-6 py-4 border-b border-slate-700 shrink-0">
          <h2 className="text-white text-lg font-bold">Refunds</h2>
          <button
            onClick={onClose}
            aria-label="Close"
            className="w-8 h-8 flex items-center justify-center text-slate-400
              hover:text-white transition-colors cursor-pointer"
          >
            <X size={18} aria-hidden="true" />
          </button>
        </div>

        {/* Body */}
        <div className="flex-1 overflow-auto p-6">
          {/* Loading */}
          {loading && (
            <p className="text-slate-400 text-sm text-center py-8">Loading…</p>
          )}

          {/* Fetch error */}
          {!loading && fetchError && (
            <p className="text-red-400 text-sm text-center py-8">{fetchError}</p>
          )}

          {/* Order list */}
          {!loading && !fetchError && step === 'list' && (
            orders.length === 0 ? (
              <p className="text-slate-400 text-sm text-center py-8">No transactions to refund today.</p>
            ) : (
              <div className="space-y-2">
                {orders.map(order => (
                  <button
                    key={order.id}
                    data-testid="order-row"
                    onClick={() => handleSelectOrder(order)}
                    className="w-full text-left bg-slate-700 hover:bg-slate-600 active:scale-[0.99]
                      rounded-xl p-3 transition-all cursor-pointer"
                  >
                    <div className="flex items-center justify-between">
                      <div className="space-y-1">
                        <div className="text-white font-semibold text-sm">
                          {formatAmount(order.total_amount)}
                        </div>
                        <div className="text-slate-400 text-xs">{formatTime(order.created_at)}</div>
                      </div>
                      <PaymentBadge method={order.payment_method} />
                    </div>
                  </button>
                ))}
              </div>
            )
          )}

          {/* Order detail */}
          {!loading && !fetchError && step === 'detail' && selectedOrder && (
            <OrderDetail
              order={selectedOrder}
              onRefundClick={handleRefundClick}
              onBack={() => setStep('list')}
            />
          )}

          {/* PIN gate */}
          {step === 'pin' && (
            <PinGate
              onConfirm={handlePinConfirm}
              onCancel={handlePinCancel}
            />
          )}

          {/* Cash confirmation */}
          {step === 'cash-confirm' && selectedOrder && (
            <CashRefundConfirm
              order={selectedOrder}
              onDone={handleCashDone}
              onCancel={handleCashCancel}
              processing={processing}
              error={refundError}
            />
          )}

          {/* Card / Tab processing */}
          {step === 'processing' && (
            <ProcessingScreen error={refundError} />
          )}
        </div>
      </div>
    </div>
  )
}
```

```jsx
            onClick={clearOrder}
            className="w-full text-slate-500 hover:text-red-400 text-sm py-2
              transition-colors duration-200 cursor-pointer flex items-center justify-center gap-1"
          >
            <Trash2 size={14} aria-hidden="true" />
            Void Order
          </button>
        )}
        <button
          onClick={() => setShowRefundsModal(true)}
          className="w-full text-slate-400 hover:text-amber-400 text-sm py-2
            transition-colors duration-200 cursor-pointer flex items-center justify-center gap-1"
        >
          <RotateCcw size={14} aria-hidden="true" />
          Refund
        </button>
      </div>

      {showCashModal && (
        <CashPaymentModal
          total={total}
          onConfirm={() => { setShowCashModal(false); handleCheckout('cash') }}
          onCancel={() => setShowCashModal(false)}
        />
      )}

      {showRefundsModal && (
        <RefundsModal
          staffMemberId={staffMemberId}
          onClose={() => setShowRefundsModal(false)}
          onRefunded={() => setShowRefundsModal(false)}
        />
      )}
    </div>
  )
}
```
